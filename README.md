## About
This repository contains practice materials and examples for HTML and CSS. 
It is designed to help learners build a strong foundation and improve their web development skills.


## Topics Covered
- HTML Basics: Elements, Tags, Attributes, Headings, Paragraphs, Lists, Links, Images, Tables
- HTML Forms: Input Types, Labels, Buttons, Form Validation
- HTML Media: Audio, Video, Embeds
- CSS Basics: Selectors, Colors, Fonts, Text Styling, Backgrounds
- Box Model: Margin, Border, Padding, Width & Height
- CSS Layouts: Positioning, Flexbox, Grid
- Responsive Design: Media Queries, Mobile-first Design, Viewport Meta Tag
- CSS Transitions and Animations
- CSS Pseudo-classes and Pseudo-elements
- CSS Variables
- CSS Specificity and Inheritance
- Advanced Styling: Gradients, Shadows, Borders
- Accessibility Basics: ARIA, Semantic HTML
